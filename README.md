> 포탈을 통해 차원을 넘나들며 몬스터를 사냥하는 핵앤슬래시 3D 액션 RPG

## 📋 프로젝트 개요
- **개발 기간**: 2026.01.02 ~ 2026.01.23 (24일)
- **팀 구성**: 3인 (프로그래머)
- **플랫폼**: PC (Windows)

## 🎮 주요 기능

### 전투 시스템
- 기본 콤보 공격 및 3종 스킬 시스템 (Q/E/R)
- 광역 스킬을 활용한 다수 몬스터 섬멸
- 화려한 VFX 이펙트

### 성장 시스템
- 레벨 기반 랭크 승급 (C → B → A → S급)
- 4등급 장비 시스템 (노말/레어/유니크/레전더리)
- 즉시 장착 시스템으로 빠른 성장 체감

### 보스 전투
- 페이즈별 변화하는 공격 패턴
- 패턴 학습 및 파훼 메커니즘

## 🛠️ 기술 스택

**Engine & Language**
- Unity 3D
- C#
